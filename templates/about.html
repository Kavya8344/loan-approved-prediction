Loan Approval Prediction System

Project Overview

The Loan Approval Prediction System is a Machine Learning-based web application developed to predict whether a loan application is likely to be approved based on an applicant's financial information. The application provides instant predictions through a simple and user-friendly web interface built using Flask.

Features

  • Real-time loan approval prediction
  • Prediction history tracking
  • Input validation for all user entries
  • Responsive web interface
  • Local prediction without external APIs

Input Features

  • Annual Income
  • Credit Score
  • Requested Loan Amount
  • Years of Employment

Machine Learning Models Evaluated

  • Logistic Regression
  • Decision Tree Classifier
  • Support Vector Machine (SVM)
  • Random Forest Classifier

Technologies Used

  • Python
  • Flask
  • Scikit-Learn
  • Pandas
  • NumPy
  • Joblib
  • HTML5
  • CSS3
  • Bootstrap 5
